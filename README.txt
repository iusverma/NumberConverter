How To Run:
This project is developed using Eclipse-Mars-4.5.0. Hence kinldy load the project on eclipse and build it.
To run the end to end scenario, run NovaGalaxyTradeDealingTest.
Add new inputs to PROJECT_DIRECTORY\input\input.txt file e.g. for windows C:\Project\input\input.txt
Check results in PROJECT_DIRECTORY\output\output.txt file e.g. for windows C:\Project\output\output.txt

Assumptions:
If statement has ? at end, it is not giving you a mapping or cost estimation
A statement is giving you a mapping, if it has like "tegj is L"
A statement is giving you a cost estimation, if doesn't have ? and has more than 3 words.
